m_n_kappa.MKappaByConstantCurvature#

class m_n_kappa.MKappaByConstantCurvature(cross_section, applied_curvature, maximum_neutral_axis, minimum_neutral_axis, applied_axial_force, maximum_iterations=10, axial_force_tolerance=5, solver=<class 'm_n_kappa.solver.Newton'>)#

Bases: MKappa

computation of one Moment-Curvature-Point by fixed curvature and varying the neutral axis

New in version 0.1.0.

Parameters:

  • cross_section (Crossection) â cross-section to compute

  • maximum_neutral_axis (float) â maximum allowed neutral-axis

  • minimum_neutral_axis (float) â minimum allowed neutral-axis

  • applied_axial_force (float) â applied axial force (Default: 0.0)

  • maximum_iterations (int) â maximum allowed iterations (Default: 10) In case the given number of iterations before axial force within desired tolerance, the computation is classified as not successful and will be stopped

  • axial_force_tolerance (float) â if axial force within this tolerance the computation is terminated and classified as successful (Default: 5.0)

  • solver (Solver) â used solver (Default: Newton)

  • applied_curvature (float) â
